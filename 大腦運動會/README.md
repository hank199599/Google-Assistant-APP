### 《大腦運動會》

說明：
-------
1.完整的題目出題與答題判斷框架。  
2.完整的錯誤輸入反饋，避免發生回應問題。

簡介
-------
來自網路上或維基百科的題庫，
透過簡易選擇題或對話機制進行遊戲。
玩家可以選擇無限題目或是最多十題題目的模式。
每個題目在選擇後會判別錯誤與否並提供相關連結供參考(大多來自維基百科)。
