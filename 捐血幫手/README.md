說明
-------
使用來自[台灣血液基金會](http://www.blood.org.tw/Internet/main/index.aspx)，  
讓使用者能隨時查詢即時的血庫庫存情形。
資訊於每日整點更新。  

此外也提供查詢各縣市固定捐血站的服務。  
可透過區域查詢或是利用GPS定位取得固定捐血站資訊。
