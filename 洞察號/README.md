說明
-------
「洞察號」是一顆用於研究火星內部結構的無人著陸探測器。  
於2019年2月19日起，NASA根據洞察號無人探測器的數據發布火星每日天氣報告，提供火星氣溫、風速、氣壓等信息。  
本服務能讓你即時取得NASA所發布的最新天氣報告。
