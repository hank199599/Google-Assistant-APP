特色說明
-------
1.完整的題目出題與答題判斷框架。  
2.完整的錯誤輸入反饋，避免發生回應問題。  
3.結束畫面的題目計算。  
4.若您想更動，您只需修改題目內容即可完成作業!  
5.使用本程式者需要在程式中或Google助理說明頁面提及代碼來源!
  

簡介
-------
來自網路上或維基百科的題庫，  
透過簡易選擇題或對話機制進行遊戲。  
玩家可以選擇無限題目或是最多十題題目的模式。  
每個題目在選擇後會判別錯誤與否並提供相關連結供參考(大多來自維基百科)。
