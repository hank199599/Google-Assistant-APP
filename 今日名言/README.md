說明
-------
隨時呼叫Google助理，即可存取[《維基語錄》](https://zh.wikiquote.org/wiki/Wikiquote:%E9%A6%96%E9%A1%B5)收錄的每日名言。  
查看名言的同時也可以前往維基百科上該名人的對應頁面連結了解更多資訊。  
  
每日更新一次(以台北標準時間為基準)
