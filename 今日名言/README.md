### 《今日名言》

說明：
-------
使用來自[《維基語錄》](https://zh.wikiquote.org/wiki/Wikiquote:%E9%A6%96%E9%A1%B5)收錄的每日名言。  
可透過Google助理快速查看，  
一併提供在維基百科上該名人的對應頁面連結。  
  
本服務會每日更新一次(以台北標準時間為基準)
